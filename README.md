# 💰 Personal Expense Tracker (React SPA)

A simple and responsive **Personal Expense Tracking Web App** built using React.  
This project allows users to add, edit, delete, filter, and visualize their expenses with a clean dashboard UI.

---

## 📌 Features

- ➕ Add new expenses (title, amount, category, date, notes)
- ✏️ Edit existing expenses
- ❌ Delete expenses
- 📋 View all expenses in a clean card layout
- 🔍 Filter expenses by:
  - Category
  - Start date
  - End date
- 📊 Visual summary using charts (Recharts)
- 💾 Persistent storage using LocalStorage
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- React (Functional Components + Hooks)
- Tailwind CSS
- Recharts (Data Visualization)
- LocalStorage API
- JavaScript (ES6+)
